Tela de Login — HTML & CSS

Bem-vindo(a) ao meu projeto de Tela de Login, feito com muita atenção aos detalhes e um toque próprio de estilo.
Aqui eu pratico conceitos essenciais de HTML e CSS, como semântica e design clean.

Sobre o Projeto

Este projeto é uma interface simples e elegante de login, construída apenas com HTML e CSS.
O foco foi:

1. Criar uma experiência agradável ao usuário
2. Trabalhar com estilização moderna
3. Aprender e reforçar boas práticas de estruturação de layout
4. É uma base perfeita para evoluir futuramente para autenticação real, uso de JS e integrações com backend.

Tecnologias usadas

HTML5
CSS3

Funcionalidades

Cores e tipografia modernas
Placeholder estilizado
Botões com efeito hover

Aprendizados

Nesse projeto, pude praticar:

Organização de layout com flexbox
Customização de placeholders
Criação de formulários estilizados
Harmonização de paleta de cores
Construção de uma interface limpa e objetiva

Melhorias futuras

Adicionar validação de formulário
Criar versão dark mode
Implementar autenticação real usando JS + API
Criar animação ao enviar o formulário
